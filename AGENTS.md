# Project

Local-first Flutter Android EPUB reader with optional contextual AI features.

The application imports EPUB files into a canonical book representation,
paginates canonical content for reading, and attaches reading state,
annotations, translations, and AI artifacts to stable content identifiers.

A physical Android device is normally connected through USB.

## Start here

For each task:

1. Read the assigned task under `tasks/active/`.
2. Read `docs/context-map.md` to know refereces to other files.
4. Use relavant skill to do the task.
